Masthead Patterns

  • The Masthead block (.-masthead) is the area shown in black. White nav areas are structurally not part of the Masthead.
  • First, the most common UC patterns.
  • Then, some common website types.
  • See accompanying patterns document (MastheadPatterns.pdf) for detailed annotations.

Implementation Notes:

  • Using the Bootstrap (V 2.1.1) Grid & Float Technique (requires wrappers: .container-fluid, .row-fluid, spanX)
  • This technique is better for simple Mastheads -- tricky for complex mastheads with 2-tier nested nav and right alignments.
  • Bootstrap .nabvar requires .navbar-inner wrapper to keep cross-browser gradients from collapsing; this is documented bug-fix
  • Tested on Chrome, Safari 5.1.2, Firefox 15.0.1
  • Few minor layout bugs: searchbar issue with Full Banner Variant 1
